Simple API to get daily data Indonesian stock exchange from Pasardana.id

Get List of All Stocks

https://getstock-info.herokuapp.com/v1/stocks/getall

Notes: this will return all the data of the listed shares without any restrictions, you can limit it by providing parameters

Get List Stocks With Quuery Parameters

https://getstock-info.herokuapp.com/v1/stocks/
Parameter Description
search

type: String

Search by Code of stock
page

type: Int

|

Default: 1

this means this is the page you requested
per_page

type: Int

this means the amount of data limit you want to retrieve
sort_by

type: String

|

Default: Code

if you want to sort data by a certain order, this is a good solution

you can sort by:

Code, Name, SectorName, SubSectorName, Last, AdjustedClosingPrice, PrevClosingPrice, AdjustedOpenPrice, AdjustedHighPrice, AdjustedLowPrice, Per, Pbr, Volume, Value, Frequency, OneDay, OneWeek, OneMonth, ThreeMonth, SixMonth, OneYear, ThreeYear, FiveYear, TenYear, Mtd, Ytd, Capitalization, BetaOneYear, StdevOneYear, LastDate,
sort_direction

type: String

|

Default: asc

you can change it to desc or asc
index_id

type: Int

Filter stocks by the given index id.
sector_id

type: Int

Filter stocks by the given sector id.
subsector_id

type: Int

Filter stocks by the given subsector id.(Not Work)

Example: https://getstock-info.herokuapp.com/v1/stocks/search?page=1&per_page=5

Get List of Index
